# TEL-001: Usage Tick Ingestion and Rollups

## Goal

Ingest provider usage data into durable ticks and expose aggregated cost and token summaries for the workspace and agent views.

## Scope

- usage tick schema handling
- rollup queries
- pricing lookup strategy
- authoritative versus estimated usage markers
- API responses for usage summary panels

## Deliverables

- usage ingestion service
- pricing config structure
- aggregate queries for `1h` and `24h` windows
- tests for cost calculations and empty-state handling

## Acceptance checks

- usage ticks persist with timestamps and token counts
- aggregated results match fixture expectations
- estimated and authoritative usage can be distinguished
- `/usage` returns workspace-scoped summaries

## Dependencies

- depends on `DB-001`
- benefits from `ADP-001` to provide fixture event streams
